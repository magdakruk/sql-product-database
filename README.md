# SQL Product Database

An individual university project presenting a web application connected to a relational product database.

## Project overview

The application was created as part of a database course at the University of Warsaw. Its purpose was to store, search and display information about products through a simple web interface.

## Main features

- browsing product records,
- searching and filtering data,
- displaying information stored in a relational database,
- communication between the web interface and the SQL database.

## Technologies

- SQL
- relational databases
- CGI-based web application
- HTML
